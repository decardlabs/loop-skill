# PRD: 智能待办清单应用

## 1. 背景
用户需要一个能自动排序、智能提醒的待办清单工具，替代传统简单的 todo list。

## 2. 目标用户
- 个人用户：日常任务管理
- 团队用户：简单的团队任务协作

## 3. 核心需求

### P0（MVP 必需）
1. **任务管理 CRUD**
   - 创建任务（标题、描述、截止日期、优先级）
   - 编辑任务
   - 删除任务
   - 标记完成/未完成
   - 列表查看（按优先级/日期排序）

2. **智能排序**
   - 按紧急程度自动排序（截止日期 + 优先级）
   - 支持手动拖拽调整

3. **分类/标签**
   - 用户可自定义标签
   - 按标签筛选

### P1（重要）
4. **提醒通知**
   - 任务到期前提醒（浏览器通知/邮件）
   - 每日待办摘要

5. **团队协作**
   - 共享任务列表
   - 分配负责人
   - 评论功能

### P2（锦上添花）
6. **数据统计**
   - 完成任务趋势图
   - 效率分析

## 4. 技术约束
- Web 应用，响应式设计（PC + 移动端）
- 前后端分离
- 支持离线使用（PWA）
- 数据实时同步

## 5. 交付标准
- 所有 P0 功能可正常使用
- 核心流程有 E2E 测试
- 部署到云端可访问
